OpenAnt

Vulnerability Discovery
GitHub Repository ↗ Visit Platform ↗

OpenAnt from Knostic is an open-source LLM-based vulnerability discovery product that helps defenders proactively find verified security flaws while minimizing both false positives and false negatives. Stage 1 detects. Stage 2 attacks. What survives is real.

Installation & Usage Guide

Local Setup

Build the CLI binary (requires Go 1.25+):
cd apps/openant-cli && make build

Symlink it onto your PATH:
ln -sf "$(pwd)/apps/openant-cli/bin/openant" /usr/local/bin/openant

Configuration

Set your Anthropic API key (requires Claude Opus 4.6 model):
openant set-api-key <your-key>

Analyzing a Project

Initialize your project (e.g., Python or Go repo):
openant init <repo-url> -l go
Run the full automated vulnerability discovery pipeline:
openant scan --verify
Or run manually: openant parse -> enhance -> analyze -> verify -> build-output -> report

Extra Info & Disclaimer

  • Supported Languages: Go, Python, JavaScript/TypeScript (beta), C/C++ (beta), PHP (beta), Ruby (beta).
  • Purpose: OpenAnt is intended for defensive and research purposes only. Scan code you own or have permission to test!
  • License: Released under Apache 2 license to help open source maintainers stay ahead of attackers.
